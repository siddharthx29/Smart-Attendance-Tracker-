# 🎓 BunkerMate | Smart Attendance Tracker

**BunkerMate** is a premium, glassmorphism-inspired web application designed for students to manage their attendance efficiently. It doesn't just track your numbers; it calculates exactly how many classes you can "bunk" while maintaining your target percentage.

![BunkerMate Mockup](bunkermate_mockup_1773301529421.png)

## 🚀 Key Features

- **📊 Dynamic Dashboard**: Get an at-a-glance view of your overall attendance, top-performing subjects, and total "bunkable" classes.
- **🧮 Intelligent Bunking Calculator**: Automatically calculates if you're "Safe" to skip a class or if you're "On the edge" and need to attend more.
- **📅 Weekly Scheduling**: Input your weekly lecture hours and specific days to get hyper-accurate daily bunking estimates.
- **📜 Detailed History**: Track every "Attended" and "Missed" lecture with a monthly breakdown.
- **💾 Persistent Data**: All your data is saved locally on your browser, so you never lose your progress.
- **📱 Responsive & Premium UI**: A stunning dark-mode interface with smooth animations and a mobile-first design.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom glassmorphism design system using CSS Variables and Flexbox/Grid.
- **JavaScript (ES6+)**: Pure vanilla logic for state management and complex calculations.
- **Lucide Icons**: Clean, consistent iconography.
- **Google Fonts**: "Outfit" typography for a modern aesthetic.

## 📖 How to Use

1. **Add a Subject**: Click the "Add Subject" button and fill in the name, weekly hours, and scheduled days.
2. **Mark Attendance**: Each day, mark your subjects as "Attended" or "Missed".
3. **Check Your Status**:
   - 🟢 **Safe**: You're well above 75%. Enjoy your free time!
   - 🟡 **Warning**: You're close to the limit. Be careful!
   - 🔴 **Danger**: You're below 75%. The app will tell you exactly how many classes you need to attend consecutively to recover.
4. **View History**: Click the history icon on any subject card to see your past logs.

## 🔧 Installation / Local Setup

Since this is a static web application, no complex installation is required.

1. Clone the repository:
   ```bash
   git clone https://github.com/siddharthx29/Smart-Attendance-Tracker-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Smart-Attendance-Tracker-
   ```
3. Open `index.html` in any modern web browser.
